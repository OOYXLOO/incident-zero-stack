# Vercel Environment Template

Use this only inside the official Vercel project settings or local `.env.local` controlled by the account owner.

```text
AWS_REGION=us-east-1
INCIDENT_ZERO_TABLE=IncidentZeroCases
AWS_ACCESS_KEY_ID=<account-owner-managed>
AWS_SECRET_ACCESS_KEY=<account-owner-managed>
```

Do not commit `.env.local`.

For a final H0 submission, add a screenshot or allowed equivalent proof showing the real AWS Database resource or Vercel storage configuration, as required by the rules.
